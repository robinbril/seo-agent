## SEO Rapport - Robin Bril

### Site analyse

**Huidige URL-structuur:**
- / (homepage, single page)
- Anchors: #services, #projects, #about, #contact, #ready-to-start
- Geen aparte pagina's per agent/dienst

**Wat goed gaat:**
- Concrete USPs (ROI binnen 1 jaar, gratis intake, 2 weken delivery)
- Client logos (KLM, E-Flux, Defensie)
- Prijsindicaties zichtbaar (€5.000+)
- 20 specifieke agent-types uitgewerkt
- Calendly koppeling voor directe conversie

**Kritieke SEO issues:**
- Single page app = één URL voor alles. Google kan weinig specifieke pagina's indexeren.
- Geen blog / kennisbank
- Vercel subdomein (geen custom domein)
- Geen schemamarkup voor diensten
- Geen meta descriptions per dienst

### Quick wins

1. **Aparte dienst-pagina's maken** - "/klantenservice-ai-agent", "/order-processing-agent" etc. Elke agent verdient eigen URL met keyword-geoptimaliseerde content. Dit is de grootste win.

2. **Blogartikel "AI agent laten bouwen: kosten en wat je kunt verwachten"** - Hoge koopintentie, weinig directe concurrentie, Robin heeft de autoriteit om dit te schrijven.

3. **Case study pagina** - KLM / E-Flux case uitwerken met probleem → oplossing → resultaat. Trekt backlinks en positioneert als expert.

### 5 artikel titels om direct te schrijven

1. **"AI agent laten bouwen: wat kost het en wat levert het op?"**
   - Target: "AI agent kosten", "AI agent laten bouwen prijs"
   - Content: prijsranges per agent type, ROI-berekening, terugverdientijd
   - ~1500 woorden

2. **"Klantenservice automatiseren met AI: van 80% automatisch naar tevreden klant"**
   - Target: "klantenservice automatiseren AI", "klantenservice agent AI"
   - Content: hoe het werkt, welke tickets automatiseerbaar, implementatieproces
   - ~1200 woorden

3. **"AI agent vs. chatbot: wat is het verschil en wanneer kies je wat?"**
   - Target: "AI agent vs chatbot", "verschil AI agent chatbot"
   - Content: educatief, topfunnel, positioneert Robin als expert
   - ~1000 woorden

4. **"Order processing automatiseren voor webshops: van Shopify naar Exact in 2 weken"**
   - Target: "order processing automatiseren", "Shopify Exact Online koppeling AI"
   - Content: technisch + praktisch, specifieke integraties, tijdsbesparing
   - ~1200 woorden

5. **"Hoe bewijs je de ROI van een AI agent aan je management?"**
   - Target: "ROI AI automatisering", "AI business case maken"
   - Content: framework voor ROI-berekening, metrics, case study voorbeeld
   - ~1000 woorden

### Technische SEO tips

1. **Custom domein:** Vercel subdomein is SEO-nadelig. robinbril.nl of quotum.ai activeren.
2. **Multi-page architectuur:** Single page naar multi-page migreren voor indexeerbaarheid
3. **Service schema markup:** FAQPage + Service + Person structured data
4. **hreflang:** NL + EN versie overwegen (internationale klanten)
5. **Backlink strategie:** Bloggen op LinkedIN + gastartikelen op MKB-media (Sprout, DutchCowboys)
6. **Core Web Vitals:** Heavy React app, controleer LCP en INP scores
